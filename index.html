Proyectos Realizados

SIMEF - Sistema de Gestión de Instituto

Sistema web (SIMEF) para la gestión de inscripciones a materias y exámenes finales, desarrollado como proyecto final de la carrera de Analista de Sistemas. Tecnologías: Frontend - HTML, CSS, Bootstrap, JavaScript; Backend - Python, Django; Base de Datos - MySQL. Rol: Desarrollador Full-Stack, con contribuciones en el diseño, desarrollo del backend y frontend, y configuración del servidor. Proyecto colaborativo con compañeros de carrera. Características principales: Gestión de inscripciones a finales, acceso al backoffice. Enlace al repositorio con detalles de instalación: GitHub

La Casa de Hilos y Repuestos

Aplicación web de comercio electrónico desarrollada con Django (Python) para la gestión y venta de productos relacionados con hilos y repuestos. Tecnologías: Python 3, Django, HTML, CSS, JavaScript, SQLite. Rol: Desarrollador Individual. Responsabilidades: Desarrollo completo de la aplicación, desde la gestión de productos y el inventario hasta el diseño de la interfaz de usuario. Características principales: Gestión de productos, gestión de inventario, promoción de productos, diseño responsive. Instalación: Clonación del repositorio (Git), creación de entorno virtual (Python venv), instalación de dependencias (pip), migración de la base de datos (Django migrate), ejecución del servidor local (Django runserver). Enlace al repositorio con detalles: GitHub

Tienda de Ropa

Aplicación web de comercio electrónico desarrollada con Django (Python) para la gestión y venta de ropa. Tecnologías: Python 3, Django, Bootstrap, HTML, CSS, JavaScript, SQLite. Rol: Desarrollador Individual. Características principales: Gestión de productos, carrito de compras, gestión de pedidos, búsqueda de productos, diseño responsive. Instalación: Clonación del repositorio (Git), creación de entorno virtual (Python venv), instalación de dependencias (pip), migración de la base de datos (Django migrate), ejecución del servidor local (Django runserver). Enlace al repositorio con detalles: GitHub
